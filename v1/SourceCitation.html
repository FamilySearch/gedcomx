This document provides the semantic definition for the GEDCOM X SourceCitation data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

This data type extends the http://gedcomx.org/v1/HypermediaEnabledData data type.

Description

Represents a source citation.

Properties

property expected type description
Properties from SourceCitation
lang string The language of the citation. See http://www.w3.org/International/articles/language-tags/
value string A rendering (as a string) of a source citation. This rendering should be the most complete rendering available.
