This document provides the semantic definition for the GEDCOM X OnlineAccount data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

This data type extends the http://gedcomx.org/v1/ExtensibleData data type.

Description

An online account for a web application.
