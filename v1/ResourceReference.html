This document provides the semantic definition for the GEDCOM X ResourceReference data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

Description

A generic reference to a resource.

Properties

property expected type description
Properties from ResourceReference
resourceId string The resource id of the resource being referenced. Used as an extension attribute when resolving the resource is inconvenient.
resource anyURI The URI to the resource. For more information, see Architecture of the World Wide Web, Volume One, Section 2
