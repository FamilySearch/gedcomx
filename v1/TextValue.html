This document provides the semantic definition for the GEDCOM X TextValue data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

Description

An element representing a text value that may be in a specific language.
