This document provides the semantic definition for the GEDCOM X Note data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

This data type extends the http://gedcomx.org/v1/HypermediaEnabledData data type.

Description

A note about a genealogical resource (e.g. conclusion or source).

Properties

property expected type description
Properties from Note
subject string The subject of the note. This is a short title describing the contents of the note text.
text string The text of the note.
attribution Attribution Attribution metadata for a note.
