This document provides the semantic definition for the GEDCOM X Qualifier data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

Description

A data qualifier. Qualifiers are used to "qualify" certain data elements to provide additional context, information, or details.

Properties

property expected type description
Properties from Qualifier
(value) string The value of the qualifier. Some qualifiers may not have values, indicating that the qualifier is to be treated more like a "tag".
