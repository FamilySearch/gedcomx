This document provides the semantic definition for the GEDCOM X NameForm data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

This data type extends the http://gedcomx.org/v1/ExtensibleData data type.

Description

A form of a name.

Properties

property expected type description
Properties from NameForm
lang string The language of the conclusion. See http://www.w3.org/International/articles/language-tags/
fullText string The full text of the name form.
part NamePart The different parts of the name form.
