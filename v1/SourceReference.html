This document provides the semantic definition for the GEDCOM X SourceReference data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

This data type extends the http://gedcomx.org/v1/HypermediaEnabledData data type.

Description

An attributable reference to a description of a source.

Properties

property expected type description
Properties from SourceReference
description string A reference to a description of the source being referenced.
attribution Attribution The attribution metadata for this source reference.
qualifier Qualifier The qualifiers associated with this source reference.
