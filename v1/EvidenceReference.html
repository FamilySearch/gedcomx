This document provides the semantic definition for the GEDCOM X EvidenceReference data type for use in structured data (e.g. RDFa, Microdata, JSON-LD, etc.). This data type is specified by the GEDCOM X Conceptual Model.

This data type extends the http://gedcomx.org/v1/HypermediaEnabledData data type.

Description

A reference to a resource that is being used as evidence.

Properties

property expected type description
Properties from EvidenceReference
resource anyURI The URI to the resource being referenced as evidence.
attribution Attribution Attribution metadata for this evidence reference.
