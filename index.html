About GEDCOM X

GEDCOM X is a project that hosts a set of open specifications for exchanging genealogical data. Examples of these data include information about persons, the relationships between persons, and the records that support that information.

GEDCOM X is proudly sponsored by FamilySearch.

Specifications

The specification set is the core of the GEDCOM X project. The specifications define the types of data being exchanged, how the data is formatted, and the protocols for exchanging the data.
